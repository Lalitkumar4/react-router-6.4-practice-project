# JobRouter (React Router 6.4)

This project is a practice project of React Router 6.4

## 🛠️ Built with

- `React.js`
- `React router dom`
- `JSON Server: For Mock API`
- `Vite`

## Quick Start

```
Install dependencies:
npm install

Run React dev server: 3000
npm run dev

Run JSON server (for mock API): 5000
npm run server
```
